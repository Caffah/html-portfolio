Unordered och ordered list

  • < ul > < /ul > används för unordered list
  • Används för att göra on oordnad lista utan siffror.
  • Ska använda < li> < /li> inom ul-taggarna.
  1. < ol> < /ol> Ska användas för ordered list
  2. Används för att göra en lista i ordning, med siffror. Börjar automatiskt från 1.
  3. Ska använda < li> < /li> inom ul-taggarna.
  4. ordered List element
  1. För att börja en Ordered list från ett annat nummer än 1, tillexempel 125 skriver du följande: < ol start=125> < /ol>
