Lägga in bilder på din hemsida.

>
  • För att lägga in bilder på din hemsida måste du använda din av img-taggen.

  1. < img
  2. src ="länk till var bilden ligger på datorn."
  3. alt ="För att beskriva vad bilden innehåller för de som har synsvårigheter."
  4. width ="400" height = "auto" exempel på hur man kan ändra storleken på bilden.

Vill du skapa en klickbar länk på din bild kan du använda dig av < a-tagen innan img-tagen.

Exempel:

  1. < a href="google.com " Om du vill att sidan öppnas i ny flik använder du target = "_blank" efter.
  2. < img
  3. src ="länk till var bilden ligger på datorn."
  4. alt ="För att beskriva vad bilden innehåller för de som har synsvårigheter."
  5. width ="400" height = "auto" exempel på hur man kan ändra storleken på bilden.
