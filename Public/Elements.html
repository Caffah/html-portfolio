Elements

Void elements kan användas för att skapa en linje mellan ställena man skriver för att avskilja, då använder man sig av < hr/> (Horizontal rule)

Void elements har inte content mellan taggarna, därför kan det vara bara att göra ett / för att se det som ett avslut direkt.

Vill du skapa ett större mellanrum mellan raderna än de över kan du använda dig av < br/> (Break element) mellan texterna

Anchor Element

Anchor elements används för att tillexempel skapa en klickbar hyperlänk

För att skapa en hyperlänk skriver du följande: < a href="länk">
